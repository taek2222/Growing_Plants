# Growing_Plants
아두이노(Arduino) + 라즈베리파이(Raspberry Pi) + 스프링부트(Spring Boot) + HTML, CSS, JavaScript

<br>

## 📑 프로젝트 설명
식물을 키우고 싶은 가구들이 증가하고 있지만. 값 비싼 재배 시설, 어려운 환경 조성 등의 이유로 식물을 키우기가 어려워짐.

이러한 문제를 해결하기 위해. 식물에게 최적의 조건을 만족시켜주도록 자동 형식의 식물 재배 환경을 만듬.
